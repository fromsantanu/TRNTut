# 📖 Chapter 18: Environment Setup (Python, PyTorch, Hugging Face)

---

## 🔷 1. Why Environment Setup is Important

Before working with Transformers, we need a proper setup.

👉 Think of it like:

🟡 **Kitchen setup before cooking**

* Ingredients → libraries
* Tools → software
* Stove → Python environment

---

### 🧠 One-Line Idea

👉
“Set up the tools correctly so everything runs smoothly.”

---

## 🔷 2. What We Need

---

### 🧰 Core Tools

1. **Python** → programming language
2. **PyTorch** → deep learning framework
3. **Hugging Face Transformers** → ready-made Transformer models

---

### 💡 What Each Does

* Python → runs code
* PyTorch → builds and runs models
* Transformers → gives prebuilt models like:

  * BERT
  * GPT

---

## 🔷 3. Step 1: Install Python

---

### ✔️ Recommended Version

* Python 3.8 or above

---

### 🧠 Check Installation

Open terminal / command prompt:

```bash
python --version
```

---

👉 If version shows → installed
👉 If not → download from python.org

---

## 🔷 4. Step 2: Create Virtual Environment (Very Important)

---

### 💡 Why?

👉 Keeps your project clean
👉 Avoids conflicts between libraries

---

### 🧠 Command

```bash
python -m venv transformer_env
```

---

### Activate Environment

#### Windows:

```bash
transformer_env\Scripts\activate
```

#### Mac/Linux:

```bash
source transformer_env/bin/activate
```

---

## 🔷 5. Step 3: Install PyTorch

---

### 💡 What is PyTorch?

👉 Library to build and run neural networks

---

### 🧠 Install Command

```bash
pip install torch torchvision torchaudio
```

---

### 🧠 Verify Installation

```bash
python -c "import torch; print(torch.__version__)"
```

---

## 🔷 6. Step 4: Install Hugging Face Transformers

---

### 💡 What is this library?

👉 Provides ready-to-use Transformer models

---

### 🧠 Install Command

```bash
pip install transformers
```

---

### Optional (Recommended)

```bash
pip install datasets
pip install accelerate
```

---

## 🔷 7. Step 5: Install Jupyter Notebook (Optional)

---

### 💡 Why?

👉 Easy to write and test code

---

### Install

```bash
pip install notebook
```

---

### Run

```bash
jupyter notebook
```

---

## 🔷 8. Step 6: Test Your Setup

---

### 🧠 Simple Test Code

```python
from transformers import pipeline

classifier = pipeline("sentiment-analysis")

result = classifier("I love learning AI")

print(result)
```

---

### ✅ Expected Output

```bash
[{'label': 'POSITIVE', 'score': 0.99}]
```

---

👉 This confirms everything is working

---

## 🔷 9. What Just Happened?

---

### 💡 Explanation

* Loaded a pretrained model
* Gave input sentence
* Model predicted sentiment

---

👉 No need to train from scratch

## 🔷 10. Optional: GPU Support (Advanced)

---

### 💡 Why GPU?

👉 Faster computation

---

### Check GPU

```python
import torch
print(torch.cuda.is_available())
```

---

👉 If True → GPU available
👉 If False → CPU only

---

## 🔷 11. Common Errors and Fixes

---

### ❌ Module not found

👉 Solution:

```bash
pip install <library_name>
```

---

### ❌ Version conflict

👉 Solution:

* Use virtual environment
* Reinstall clean setup

---

### ❌ Slow execution

👉 Solution:

* Use GPU
* Reduce data size
