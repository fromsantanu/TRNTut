## 🔷 4. Step 2: Load Model and Tokenizer

---

```python id="t5a2"
from transformers import T5Tokenizer, T5ForConditionalGeneration

tokenizer = T5Tokenizer.from_pretrained("t5-small")
model = T5ForConditionalGeneration.from_pretrained("t5-small")
```

---

## 🔷 5. Project 1: Text Translation

---

### 🧠 Example Task

👉 English → Hindi

---

### 🧠 Code

```python id="t5a3"
text = "translate English to Hindi: I am happy"

inputs = tokenizer(text, return_tensors="pt")
outputs = model.generate(**inputs)

result = tokenizer.decode(outputs[0], skip_special_tokens=True)

print(result)
```

---

### ✅ Output (Example)

👉 “मैं खुश हूँ”

---

## 🔷 6. Project 2: Text Summarization

---

### 🧠 Example Input

👉
“The movie was very long but it had an interesting story and strong characters.”

---

### 🧠 Code

```python id="t5a4"
text = "summarize: The movie was very long but it had an interesting story and strong characters."

inputs = tokenizer(text, return_tensors="pt", max_length=512, truncation=True)
outputs = model.generate(inputs["input_ids"], max_length=20)

summary = tokenizer.decode(outputs[0], skip_special_tokens=True)

print(summary)
```

---

### ✅ Output (Example)

👉 “Interesting but long movie”

---

## 🔷 7. How It Works Internally

---

1. Input text is tokenized
2. Encoder understands meaning
3. Decoder generates output
4. Output converted back to text

---

👉 Full encoder–decoder process

---

## 🔷 8. Improving Results

---

### 💡 Tips

* Use better prompts
* Adjust `max_length`
* Try larger models (`t5-base`, `t5-large`)
* Clean input text

## 🔷 9. Advanced Control

---

### 🧠 Example

```python id="t5a5"
outputs = model.generate(
    inputs["input_ids"],
    max_length=30,
    num_beams=4,
    early_stopping=True
)
```

---

### 💡 Parameters

* `num_beams` → improves quality
* `max_length` → controls output size
