Text generation means:

👉 The model predicts **next word → then next → then next**

Input:

## 🔷 4. Basic Text Generation (Pipeline Method)

---

### 🧠 Code Example

```python id="gpt1"
from transformers import pipeline

# Load text generation pipeline
generator = pipeline("text-generation")

# Generate text
result = generator("Once upon a time", max_length=30)

print(result)
```

---

### ✅ Output (Example)

```bash id="gpt2"
Once upon a time, there lived a brave king who ruled the land with wisdom.
```

---

## 🔷 5. Understanding the Code

---

### 💡 What happens?

* Loads pretrained GPT model
* Takes input text
* Predicts next words step-by-step

---

## 🔷 6. Controlling Output

We can control how the model generates text.

---

### 🧠 Example

```python id="gpt3"
result = generator(
    "The future of AI is",
    max_length=40,
    num_return_sequences=2
)

print(result)
```

---

### 💡 Parameters

* `max_length` → total words generated
* `num_return_sequences` → number of outputs

---

## 🔷 7. More Control (Temperature & Creativity)

---

### 🧠 Code

```python id="gpt4"
result = generator(
    "The future of AI is",
    max_length=40,
    temperature=0.7
)
```

---

### 💡 What is Temperature?

* Low (0.2) → predictable
* Medium (0.7) → balanced
* High (1.0+) → creative

---

### 🟡 Analogy

* Low → textbook answer
* High → creative storytelling
