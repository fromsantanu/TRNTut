## 🔷 5. Using Hugging Face Pipeline (Easiest Way)

---

### 🧠 Code Example

```python id="n7g8mz"
from transformers import pipeline

# Load pretrained model
classifier = pipeline("sentiment-analysis")

# Input text
result = classifier("I love learning AI")

print(result)
```

---

### ✅ Output

```bash id="dx3z4g"
[{'label': 'POSITIVE', 'score': 0.99}]
```

---

## 🔷 6. What is Happening Behind the Scene?

---

### Step-by-Step

1. Model loads pretrained BERT
2. Text is converted into tokens
3. Model analyzes context
4. Output label is predicted

---

👉 All done in one line using `pipeline`

## 🔷 8. Using BERT More Directly (Advanced Step)

---

### 💡 Instead of pipeline, use model + tokenizer

---

### 🧠 Code

```python id="4xj3m2"
from transformers import AutoTokenizer, AutoModelForSequenceClassification
import torch

# Load model and tokenizer
tokenizer = AutoTokenizer.from_pretrained("bert-base-uncased")
model = AutoModelForSequenceClassification.from_pretrained("bert-base-uncased")

# Input text
inputs = tokenizer("I love AI", return_tensors="pt")

# Model prediction
outputs = model(**inputs)

print(outputs.logits)
```

---

👉 This gives raw output (numbers)
