# 📖 Chapter 21: Text Classification Project

## 🔷 1. Project Goal

Build a simple **text classification system** that can label text into categories.

👉 We’ll use an encoder model like BERT

---

### 🎯 Objective

👉 Given a sentence → predict its category

---

### 🧠 One-Line Idea

👉
“Read the text → understand it → assign a label.”

---

## 🔷 2. What is Text Classification?

Text classification means:

👉 **grouping text into predefined categories**

---

### 🧠 Examples

| Input Text            | Output   |
| --------------------- | -------- |
| “I love this product” | Positive |
| “This is terrible”    | Negative |
| “Win money now!!!”    | Spam     |

---

## 🔷 3. Project Type

We will build:

👉 **Sentiment Classifier (Positive / Negative)**

---

## 🔷 4. Step 1: Install Required Libraries

```bash
pip install transformers datasets torch
```

---

## 🔷 5. Step 2: Load Dataset

We’ll use a simple dataset (can also use your own).

---

### 🧠 Example Code

```python
from datasets import load_dataset

dataset = load_dataset("imdb")

print(dataset)
```

---

👉 This dataset contains movie reviews:

* Positive
* Negative

---

## 🔷 6. Step 3: Load Tokenizer

---

### 💡 Tokenizer converts text → numbers

```python
from transformers import AutoTokenizer

tokenizer = AutoTokenizer.from_pretrained("bert-base-uncased")
```

---

## 🔷 7. Step 4: Tokenize Data

---

```python
def tokenize(example):
    return tokenizer(example["text"], truncation=True, padding="max_length")

tokenized_dataset = dataset.map(tokenize, batched=True)
```

---

## 🔷 8. Step 5: Load Model

---

```python
from transformers import AutoModelForSequenceClassification

model = AutoModelForSequenceClassification.from_pretrained(
    "bert-base-uncased",
    num_labels=2
)
```

---

## 🔷 9. Step 6: Training Setup

---

```python
from transformers import TrainingArguments

training_args = TrainingArguments(
    output_dir="./results",
    learning_rate=2e-5,
    per_device_train_batch_size=8,
    num_train_epochs=1
)
```

---

## 🔷 10. Step 7: Train Model

---

```python
from transformers import Trainer

trainer = Trainer(
    model=model,
    args=training_args,
    train_dataset=tokenized_dataset["train"],
    eval_dataset=tokenized_dataset["test"]
)

trainer.train()
```

---

## 🔷 11. Step 8: Test the Model

---

```python
result = trainer.predict(tokenized_dataset["test"])
print(result)
```

---

## 🔷 12. Step 9: Simple Prediction

---

```python
text = "This movie was amazing!"

inputs = tokenizer(text, return_tensors="pt", truncation=True)
outputs = model(**inputs)

print(outputs.logits)
```

---

👉 Convert logits to label:

* Positive or Negative

---

## 🔷 13. What is Happening Internally?

---

1. Text → tokenized
2. Converted into embeddings
3. Passed through BERT encoder
4. Classification layer predicts label
